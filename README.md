# Simple Kanban Board

A lightweight, browser-based Kanban board to help you track and manage your daily tasks using basix HTML5, CSS3 and Javascript

## Features

* **Task Creation:** Quickly add new tickets to your workflow.
* **Progress Tracking:** Move cards through three standard agile stages: *To Do*, *In Progress*, and *Done*.
* **Click-to-Move Interaction:** Simply click a card to advance it to the next column.
